HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element displays an image on your web page allowing you to assign a source for the impage and a caption.
  • The section element defines different content sections on your webpage.
  • The meta element determines invisibile characterstics of your webpage that affects user experience.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class relates to a set of CSS atributes given to an assignable declaration for whatever elments we choose
  • It is typically best to have a seperate CSS sheet from your HTML file. If you use an external sheet however you have to remember to go back and link it to the html file.

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push origin "Branch you want to push to": pushs code changes to online Git repository
  • git pull origin "Branch you want to pull": pulls any new changes made by others to the branch you are working on
